# MusicToggle
Cookie Clicker mod to toggle the music in background using M when tabbed in or Ctrl+M when tabbed out!

Just place the file in \Cookie Clicker\ and run the python or application, it will patch the start.js for the needed api addition and install the mod to the users local mods directory!

The application is simply the python file named ModInstaller packed with pyinstaller, for clean and pythonless installation, if you'd like to be sure of the authenticity the user can always use the source code directly!

I mainly made this for the clicker clicker game like the steam edition cause I wanted to turn the background music on and off quickly without navigating the menus!
